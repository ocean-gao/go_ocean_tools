# go_ocean_tools

基于 go 的小工具集合

## 1. 相关链接

- [color](https://github.com/fatih/color)
- [cobra](https://github.com/spf13/cobra)
- [cobra-cli](https://github.com/spf13/cobra-cli)

## 2. 初始化

```bash
cd go_ocean_tools
go mod init github.com/ocean-gao/go_ocean_tools

go install github.com/spf13/cobra-cli
cobra-cli init
```

运行：

```bash
go run main.go
```
